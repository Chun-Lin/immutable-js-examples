# This is a project demonstrating immutable JS API examples.

> All examples can be found in `index.js`

## Prerequisite
1. Node.js
2. [Immutable.js Object Formatter](https://chrome.google.com/webstore/detail/immutablejs-object-format/hgldghadipiblonfkkicmgcbbijnpeog)

## Steps to execute the project and see the results
1. `yarn`
2. `yarn start`
3. Open DevTools to check the results
